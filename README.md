# CRNN_kreas

**文本识别分为两部分：文本定位与文本序列识别。这个repo主要是做的后者。**

这是一个基于CRNN的文本序列识别项目，目前正在测试数字串的识别。之后会加入更多的文字识别。

其中src文件夹下面存储的是源码，data下面存储的是数据。

持续更新中。。。
